Why CNN for Image Classification?

In machine learning, Convolutional Neural Networks (CNN or ConvNet) are complex feed forward neural networks. CNNs are used for image classification and recognition because of its high accuracy.

It was proposed by computer scientist Yann LeCun in the late 90s, when he was inspired from the human visual perception of recognizing things.

  • There are a lot of algorithms that people used for image classification before CNN became popular. People used to create features from images and then feed those features into some classification algorithm like SVM.
  • Some algorithm also used the pixel level values of images as a feature vector too. To give an example, you could train a SVM with 784 features where each feature is the pixel value for a 28x28 image.

    • So why CNN and why they work so much better?

  • CNNs can be thought of automatic feature extractors from the image.
  • While if I use a algorithm with pixel vector I lose a lot of spatial interaction between pixels, a CNN effectively uses adjacent pixel information to effectively downsample the image first by convolution and then uses a prediction layer at the end.

VGG19 Architecture

The input to VGG based convNet is a 224*224 RGB image. Preprocessing layer takes the RGB image with pixel values in the range of 0–255 and subtracts the mean image values which is calculated over the entire ImageNet training set.

  • A fixed size of (224 * 224) RGB image was given as input to this network which means that the matrix was of shape (224,224,3).
  • This research aspires to develop a Deep-Learning System (DLS) to diagnose the lung abnormality using chest X-ray (radiograph) images. The proposed work is implemented using; (i) Conventional chest radiographs and (ii) Chest radiograph treated with a threshold filter.
  • The initial experimental evaluation is carried out using the traditional DLS, such as AlexNet, VGG16, VGG19 and ResNet50 with a SoftMax classifier. The results confirmed that, VGG19 provides better classification accuracy (86.97%) compared to other methods.
  • The result confirms that VGG19 with RF classifier offers better accuracy (95.70%). When the similar experiment is repeated using threshold filter treated chest radiographs, the VGG19 with RF classifier offered superior classification accuracy (97.94%).
  • This result confirms that, proposed DLS will work well on the benchmark images and in the future, it can be considered to diagnose clinical grade chest radiographs.

VGGNet is a Convolutional Neural Network architecture proposed by Karen Simonyan and Andrew Zisserman from the University of Oxford in 2014. This paper mainly focuses on the effect of the convolutional neural network depth on its accuracy.

Why we used transfer learning

In the context of deep learning, transfer learning implies a transference (or reuse) of knowledge which was used to train a model for one task to train another model for other task.

Through this method, the more generic layers of the model such as the ones closer to the input are kept as is, whereas the task-specific layers (adapted for task X or Y) which are closer to the final output layers are modified. A few things to keep in mind are:

  • Task X and task Y are relatively similar
  • Datasets for Task X and Task Y are relatively similar. Of course, there are cases when the datasets for Task X and Task Y are widely different, but in these cases, the size of the dataset is helpful to make the decision.

Why is Transfer Learning popular in Deep Learning?

  • Learning capability: There is a good transferability of knowledge amongst neural networks. Several models for several tasks have been successfully trained using transfer learning. This is a good paper to read: How transferable are features in deep neural networks?
  • Feature Extraction: Several of the pre-trained models are often utilized as a feature extractor. These features once extracted are then fed to further downstream tasks through other task-specific layers (eg ImageNet trained classification to Dog breed classification) or another learning algorithm (eg ImageNet trained classification to image captioning models). This builds into Multi-Task Learning.
  • Data: Deep Learning neural networks have been empirically shown to be data-hungry and usually require “big data”. However, there are a lot of tasks that don’t have a lot of data or don’t have a lot of labeled data available. Transfer learning enables fine-tuning a pre-existing model to work for such tasks.
